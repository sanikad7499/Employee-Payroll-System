#  Employee Payroll & Workforce Analytics System

![Python](https://img.shields.io/badge/Python-3.x-blue?logo=python)
![NumPy](https://img.shields.io/badge/NumPy-Library-013243?logo=numpy)
![Status](https://img.shields.io/badge/Status-Completed-brightgreen)
![License](https://img.shields.io/badge/License-MIT-yellow)

A **Python-based Payroll Management System** that streamlines employee record management while leveraging **NumPy** for workforce analytics. This project demonstrates **Object-Oriented Programming (OOP)**, **File Handling**, **Exception Handling**, and **Numerical Analysis** through a practical payroll application.

---

##  Features

*  Add employee details (ID, Name, Department, Salary)
*  Store and retrieve records using text files
*  Object-Oriented Employee class implementation
*  Validate user inputs using exception handling
*  Salary analytics with NumPy

  * Mean
  * Median
  * Standard Deviation
  * Highest Salary
  * Lowest Salary
*  Apply salary increments using NumPy broadcasting
*  Department-wise matrix operations
*  Generate random employee datasets for testing
*  Filter employees using Boolean Indexing
